Serif

Serif is an expressive, practical functional programming language.

Serif is dynamically typed and strictly evaluated.

Serif code most closely resembles Haskell code, but the experience of programming with Serif feels light in some indescribable way.

Serif is fully interoperable with JavaScript: Serif modules can import ES modules; Serif modules are transpiled to ES modules that can be imported by ES modules written in JavaScript or TypeScript.

Array literals

Array literals begin with [ and end with ], and contain zero or more elements separated by ,. A trailing comma is permitted.

Object literals

{}
{[key]: value}
{["x"]: 1, ["y"]: 2}
{x: 1, y: 2}
{x: 1, y: 2,}
{foo, bar, baz}
{foo, bar}
{foo,}

Object literals begin with { and end with }, and contain zero or more entries separated by ,. An entry is a key and a value separated by :. The most flexible form permits an expression to appear between [ and ] to determine the property name dynamically. {x: 1, y: 2} is shorthand for {["x"]: 1, ["y"]: 2}. {foo, bar} is shorthand for {foo: foo, bar: bar}. A trailing comma is permitted, and required in the case of {foo}—an object with a single shorthand entry—to avoid being interpreted as a block expression.
